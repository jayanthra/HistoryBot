# HistoryBot


Twitter bot that tweets what happened that day in history.


# Demo

https://twitter.com/jayanthisabot

### Installation

```sh
$ npm install
```
Enter your keys in the config file 

```javascript
module.exports = {
  consumer_key: '',
  consumer_secret: '',
  access_token: '',
  access_token_secret: '',
}
```
### Heroku setup

git push heroku master
